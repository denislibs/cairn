### Task 1: `createRipple`
- Files: new `packages/material/src/ripple.ts` (`createRipple`, `RippleHandle`), export from `packages/material/src/index.ts`. Test `packages/material/test/ripple.test.ts`.
- `createRipple(opts?)`: signal `ripples: {id;x;y;p:()=>number}[]`; `trigger(x,y)`: push `{id, x, y, p:getP}` with a `[getP,setP]` signal, `animate({from:0,to:1,duration:opts?.duration??450,easing:'ease-out',onUpdate:setP,onDone:()=>removeById(id)})` (from `@cairn/runtime`); `instance` = raw Instance with `layout: new BoxNode({ width:'100%', height:'100%' })`, `paintSelf(r)`:
```ts
    const w = layout.size.w, h = layout.size.h;
    const list = ripples();
    if (!list.length) return;
    r.save();
    r.clipRoundRect({ x:0, y:0, width:w, height:h }, opts?.radius ?? 0);
    for (const rp of list) {
      const p = rp.p();
      const maxR = Math.hypot(Math.max(rp.x, w - rp.x), Math.max(rp.y, h - rp.y));
      const R = maxR * p;
      r.fillRoundRect({ x: rp.x - R, y: rp.y - R, width: R*2, height: R*2 }, R, { color: alpha(color, 0.3 * (1 - p)) });
    }
    r.restore();
```
  (`color = opts?.color ?? 'rgba(0,0,0,1)'`; `alpha` from `./colors`.) `onCleanup` cancels active ripple animations (track cancels in a map).
- TDD (fake host manual clock + recording renderer): before trigger, `paintSelf` draws nothing; after `trigger(10,10)` + setting `instance.layout.size`, `paintSelf` calls `fillRoundRect`; ticking the clock advances progress (radius grows — arg differs) and after full duration the ripple is removed (`paintSelf` draws nothing again).
